# In Progress

仍在开发中的 skills。它们尚未准备好发布——预期会有粗糙之处、破坏性变更以及被弃置的实验。在它们晋升到稳定 bucket 之前，会被排除在插件和顶层 README 之外。

- **[review](./review/SKILL.md)** — 沿着两条平行的轴审查自固定点以来的变更：**Standards**（diff 是否遵循该仓库的编码标准？）和 **Spec**（diff 是否如实实现了起源的 issue/PRD？）。
- **[writing-beats](./writing-beats/SKILL.md)** — 以"节拍（beats）之旅"的方式塑造一篇文章，采用选择你自己冒险（choose-your-own-adventure）的风格。挑一个起始节拍，仅写那个节拍，然后转向下一个，直到文章自然收尾。
- **[writing-fragments](./writing-fragments/SKILL.md)** — 盘问环节，从你身上挖掘片段——异质的写作小块——并将它们追加到单一文档中，作为未来文章的原材料。
- **[writing-shape](./writing-shape/SKILL.md)** — 取一份原材料 markdown 文件，逐段塑造成一篇文章，在每一步都论证格式选择。
